# User Registration System (Interview Task)

## Overview

A simple PHP-based user registration system developed as an interview task ~3 years ago.
Includes basic form handling, database operations, and email integration.

---

## Features

* User registration (AJAX)
* Email uniqueness check
* Password hashing
* Email notifications
* User listing
* Export to Excel

---

## Tech Stack

* PHP (OOP)
* MySQL (PDO)
* JavaScript (AJAX)
* PHPMailer

---

## Notes

This project reflects my early development stage.
If rebuilt today, I would improve validation, security, and overall architecture (e.g. better separation of concerns, environment configs, and safer password handling).
